JavaScript : Nested Function

Nested Function

The arrw function is one of the features introuced in the ES6 version of JavaScript. It allows you to create functions in a cleaner way compared to regular functions.
Here are some of the valid syntax for arrow function:

Syntax for Arrow function: 
  1.                         () => expression
                    
  2.                         (param1, paramN) => expression
                    
  3.                         () => {
                            statements
                        }
                    
  4.                         (param1, paramN) => {
                            statements
                        }
                    
            
Example:

            //example : 1
            const greet = (count) => {
                for (let index = 0; index < count; index++) {
                    console.log("Hello World!")
                }
            }
            
            console.log(greet(3));
            
Output:

            25
