JavaScript : Nested Function

Nested Function

Functions within another function are called “Nested function.” A function can have one or more inner functions. These nested functions (inner functions) are under the outer functions’ scope. In Javascript Nested Functions, the outer function can be called the Parent function, and the inner function can be called the Child function. The child function can access the variables and parameters of the Parent function.

Approach

  1. Write one function inside another function. (square fn inside addSquare fn)
  2. Make a call to the inner function in the return statement of the outer function.
  3. Call it addSquare(val1, val2), where a=3 and b=4 is the parameter to the outer function.
  4. Finally, return the combined output from the nested function.
Syntax for Function Declaration: 
            function parent_name ( param1, param2 ){
                function child_name ( param3, param4 ){
                   statements; //body of the child function
                }
                statements; //body of the parent function
                child_name() //calling of child function
             }
             parent_name() //calling of parent function
            
            
Example:

            //example : using function declaration
            function addSquare(val1, val2) {

                const squareOfVal1 = square(val1);
                const squareOfVal2 = square(val2);
            
                function square(num) {
                    return num * num;
                }
            
                return squareOfVal1 + squareOfVal2;
            }
            
            console.log(addSquare(4, 3)); 
            
Output:

            25
