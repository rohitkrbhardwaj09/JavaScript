JavaScript : Function Expression

Function Expression

The Javascript Function Expression is used to define a function inside any expression. The Function Expression allows us to create an anonymous function that doesnâ€™t have any function name which is the main difference between Function Expression and Function Declaration.

A function expression can be used as an IIFE (Immediately Invoked Function Expression)which runs as soon as it is defined. A function expression has to be stored in a variable and can be accessed using variableName.

With the ES6 features introducing Arrow Function, it becomes more easier to declare function expression.

Syntax for Function Declaration:

function functionName(x, y) { statements... return (z) };

            
Example:

            //example : using function declaration
            function square(num){
                return num*num;
            }
            
            console.log(square(3));
            
Output:

            9
Syntax for Function Expression (anonymous):

let variableName = function(x, y) { statements... return (z) };

            
Example:

            //example : using function expression
            const square = function(num){
                return num*num;
            }

            console.log(square(3)); 
            
Output:

            9
Syntax for Function Expression (named):

let variableName = function functionName(x, y) { statements... return (z) };

            
Example:

            //example : using function expression
            const square = function calSquare(num){
                return num*num;
            }

            console.log(square(3)); 
            
Output:

            9
Syntax for Arrow Function:

let variableName = (x, y) => { statements... return (z) };

            
Example:

            //example : using function expression
            const square = (num) => {
                return num*num;
            }

            console.log(square(3)); 
            
Output:

            9
