JavaScript : Arrays

Arrays in JavaScript

JavaScript Array is a single variable that is used to store elements of different data types. JavaScript arrays are zero-indexed. The Javascript Arrays are not associative in nature.

Arrays are used when we have a list of items. An array allows you to store several values with the same name and access them by using their index number.

Declaration of an Array:

There are basically two ways to declare an array.

1. Creating an array using array literal:
let arrayName = [value1, value2, ...]; 
            
Example:

            //Creating an array using array literal
            let courses = ["HTML", "CSS", "JavaScript"];
            console.log(courses);
            
Output:

            (3) ['HTML', 'CSS', 'JavaScript']
2. Creating an array using the JavaScript new keyword:
let arrayName = new Array(); 
            
Example:

            //Creating an array using the Javascript new keyword
            let arr1 = new Array(3);
            arr1[0] = 10;
            arr1[1] = 20;
            arr1[2] = 30;

            console.log(arr1);
            
Output:

            Array(3)
        
            
Another Example:

            //Initializing while declaring
            let arr2 = new Array(10, 20, 30, 40, 50);
            
            console.log(arr2);
            
Output:

            (5) [10, 20, 30, 40, 50]
        
            
Another Example:

            //Creates an array of 5 undefined elements
            let arr3 = new Array(5);
            
            console.log(arr3);
            
Output:

            (5) [empty x 5]
        
            
Another Example:

            //Creates an array with one element
            let arr4 = new Array("1BHK");

            console.log(arr4);
            
Output:

             ['1BHK']
Accessing Elements of an Array

Any element in the array can be assessed using the index number. The index in the arrays starts with 0. 

            
Example:

            //Accessing array element using index number
            const courses = ["HTML", "CSS", "JavaScript"];
            console.log(courses[0]);
            console.log(courses[1]);
            console.log(courses[2]);

            
Output:

             HTML
             CSS
             JavaScript
Change elements from a pre-defined array

We will use index based method to change the elemetns of array. 

            
Example:

            const courses = ["HTML", "CSS", "JavaScript"];
            console.log(courses[0]);
            console.log(courses[1] = "NODE Js");
            console.log(courses[2]);

            
Output:

             HTML
             NODE Js
             JavaScript
Convert an array to String

We have an in-built method toString() in JavaScript that converts an array to String 

            
Example:

            // Convert an array to String 
            const courses  = ["HTML", "CSS", "JavaScript"];

            console.log(courses.toString());

            
Output:

             HTML,CSS,JavaScript
Increase and decrease the lengh of an array

We can increase and decrease the length of an array using the javascript's length property. 

            
Example: Increasing

            //Increasing the length of the array  
            const courses  = ["HTML", "CSS", "JavaScript"];
            courses.length = 5; //Increasing array length to 5

            console.log(courses);
            
Output:

             (5) ['HTML', 'CSS', 'JavaScript', empty x 2]
        
            
Example: Decreasing

            //Increasing the length of the array  
            const courses = ["HTML", "CSS", "JavaScript"];
            courses.length = 2; //Decreasing array length to 2 

            console.log(courses);
            
Output:

             (2) ['HTML', 'CSS']
We can also update an array after initialization
            
Example: 

             
            const courses = ["HTML", "CSS", "JavaScript"];
            courses.length = 5;
            console.log("Array after increased length: ",courses);

            courses[3] = "Php";
            courses[4] = "React js";
            console.log("Array after re-initializing: ", courses);
            
Output:

            Array after increased length: 
            (5) ['HTML', 'CSS', 'JavaScript', empty x 2]

            Array after re-initializing: 
            (5) ['HTML', 'CSS', 'JavaScript', 'Php', 'React js']
Loop through JavaScript Array Elements
For Loop

We can loop through the elements of a JavaScript array using the for loop: 

            
Example: 

            //Iterating array using for loop 
            const courses  = ["HTML", "CSS", "JavaScript"];
            for (let i = 0; i < courses.length; i++) {
                const element = courses[i];
                console.log(element);
            }
            
Output:

            HTML
            CSS
            JavaScript
forEach Loop

We can loop through the elements of a JavaScript array using the for loop: 

            
Example: 

            //Iterating array using Array.forEach() loop 
            const courses  = ["HTML", "CSS", "JavaScript"];
            
            courses.forEach(myFucntion);
            function myFucntion(element){

                console.log(element);
            
            }
            
Output:

            HTML
            CSS
            JavaScript
Adding new elements to JavaScript Array

Using the Javascript built-in push() method we can add new elements to an array.

            
Example: 

             
            const courses  = ["HTML", "CSS", "JavaScript"];
            console.log("Original Array: ", courses);
            courses.push("React");
            console.log("Array after adding an element: ", courses);
            
Output:

            Original Array:
            (3) ['HTML', 'CSS', 'JavaScript']

            Array after adding an element:
            (4) ['HTML', 'CSS', 'JavaScript', 'React']

We can also add a new element to a JavaScript Array using the length property.

            
Example: 

            // Adding element to an array using length property 
            const courses  = ["HTML", "CSS", "JavaScript"];
            console.log("Original Array: ", courses);
            courses[courses.length] = "React";
            console.log("Array after adding an element: ", courses);
            
Output:

            Original Array:
            (3) ['HTML', 'CSS', 'JavaScript']

            Array after adding an element:
            (4) ['HTML', 'CSS', 'JavaScript', 'React']
