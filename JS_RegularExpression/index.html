Regular Expression in JavaScript

A Regular Expression is a sequaence of character that forms a seach pattern. The search pattern can be used for text search and text replace operations.
A Reguar Expression can be a single character or a more complicated pattern. Regular Expression can be used to perform all types of text search and text replacement operations.

Syntax:
/pattern/modifier 
            
Example:

            
            let pattern = /rohitbhardwaj9/i;
            
Explanation:

            
rohitbhardwaj9 is a regular expression. 
rohitbhardwaj9 is the pattern (to be used in a search). 
i is a modifier (modifies the seach to be Case-Insensitive)
Regular Expression Modifiers :
can be used to perform multiline searches which can also be set to Case-Insensitive matching.
Expression Description
i Find a character with Case-Insensitive matching
g Find the character globally
m Find multiple matching
Regular Expression Brackets :
can be Find characters in a specified range
Expression Description
[abc] Find any of the characters inside the brackets
[^abc] Find any character, not inside the brackets
[0-9] Find any of the digits between the brackets 0 to 9
[^0-9] Find any digit not in between the brackets
(x|y) Find any of the alternatives between x or y separated with |

JavaScript Regular Expression

  • A Regular Expression is a String that describes a pattern e.g. email addresses and phone numbers.
  • In JavaScript, Regular Expression are objects. JavaScript provides the built-in RegEx type that allow you to work with regular expressions effectively.
  • Regular Expression are useful for searching and replacing string that match a pattern. They have many usefull applications.

Creating a Regular Expression

To create a regular expression in JavaScript, you enclose its pattern in forward-slash characters (/) like this

let re = /hi/;
Note that a Regular Expression doesn't have single quotes or double quotes like a regular string.

Or you can use the RegExp constructor

let re = new RegExp('hi'); 
            
Testing for matching example: using test() method

            //Creating a regular expression
            
            let re = /hi/;
            // or 
            let re = new RegExp('hi');

            let result = re.test('hi jhon');
            console.log(result);
            
Output:

            true

Using pattern flags

Besides apattern, a RegExp object also accepts an optional flag parameter. Flags are setting that change the searching behavior. Regular espression have many flags.

1) The ignore flag (i)

The i flag ignores cases when searching. The letter i stands for ignore.

By default, searches are case-sensitive. For Example hi only matches the string hi not Hi. To search for a string hi in any cases, you use the i flag: 

            
Example: 

            let re = /hi/i;
            let result = re.test("Hi Jhon");

            console.log(result);
            
Output:

            true

Or you can use the RegExp constructor

hi /hi/i will match any string hi, Hi, and HI.
Notice that you place the flag i after the last forward-slash character / 
            
The follwoing examples shows how to use the flag in the RegExp constructor: 

            //Creating a regular expression
            
            let re = new RegExp('hi', 'i');
            let result = re.test('HI Jhon');

            console.log(result);
            
Output:

            true

2: The global flag (g)

Another commonly used flag is the global flag (g). Without the global flag, the RegExp object only checks if there is a match in a String and returns the first match.

When the g flag is available, the RegExp looks for all the matchs and returns all of them.

It's possible to combine flags e.g. gi flags combine the ignore (i) and the gloabal (g) flags

The exec() method returns null if it could not find any match. However, it returns a single match at once. To get all matches, you need to execute the exec() multiple times. 

            
The following exapmle uses the exec() method with a do...while loop to return all the matches:  

             // The global case
        
            let message = 'Hi, are you there? hi, HI...';
            let re = /hi/gi;
            
            let matches = [];
            let match;
            
            do {
                match = re.exec(message);
                if (match) {
                    matches.push(match);
                }
            } while (match != null);
            
            console.log(matches);
            
Output:

            ...
How it works:
  • First, declare a message string that will be used for searching.
  • Then, create a regular expression object with the pattern /hi/gi. The ignore flag (i) allows re object to ignore cases when executing the search and global flag (g) instructs the re object to find all matches, not just the first one.
  • Third, execute the exec() method until no match found.
  • Finally, show the result array in the console.

Character Classes

Introduction to the character class

A character class allows you to match any symbol from a certain character set. A character class is also called a character set. Suppose that you have a phone number like this:

+1-(408)-555-0105

And you want to turn it into a plain number.

14085550105

Character classes in regular expressions can help you to do this.

Digit character

Let’s explore the digit character class first. The digit character class is denoted by \d which matches any single digit:

\d

The following example uses the \d to match the first number in the phone number: 

            
Example: 

            // digit character
            let phone = '+1-(408)-555-0105';
            let re = /\d/;
            console.log(phone.match(re));
            
            
Output:

                1

When you add the global flag (g), the regular expression will search for all numbers, not the first one

            
Example: 

            //Using global flag
            
            let phone = '+1-(408)-555-0105';
            let re = /\d/g;

            console.log(phone.match(re));
            
            
Output:

                (11) ['1','4','0','8','5','5','5','0','1','0','5']

Now, you can turn the phone number into a plain number as follows:

  • Use the match() method to return an array containing numbers.
  • Use the join() method to concatenate elements of the array into a string.

            
Example: 

            //Using Join()
            
            let phone = '+1-(408)-555-0105';
            let re = /\d/g;
            let numbers = phone.match(re);
            let phoneNo = numbers.join('');
            
            console.log(phoneNo);
            
            
Output:

                14085550105
To make it short, you can chain the match() and join() methods like this:
console.log('+1-(408)-555-0115'.match(/\d/g).join(''));

Besides the digit character class (\d), regular exprssions support other character classes.

The most commonly used character classes are:

  • \d - match a single digit or a character from 0 to 9.
  • \s - match a single whitespace symbol such a space, a tab (\t), a newline (\n)
  • \w - w stands for word character. It matches the ASCII character [A-Za-z0-9] including latin alphabets, digits, and the underscore (_)

Space character : (\s)

The \s metacharacter matches whitespace character. Whitespace characters can be:

  • A space character
  • A tab character
