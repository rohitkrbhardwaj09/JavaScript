The at() method

The at() method returns an indexed element from an array. 

            
Example:

            //Trying to get element from 2nd indexed from the fruit named array
            const fruit = ["Apple", "Banana", "Cherry"];
            console.log(fruit.at(2));

            
Output:

            Cherry
JavaScript Array concat() method

The concat() method concatenates (joins) two or more arrays: 

            
Example:

            //Concatanating arr1 with arr2
            const arr1 = ["One", "Two", "Three", "Four"];
            const arr2 = ["Five", "Six", "Seven"];

            const result = arr1.concat(arr2);
            console.log(result);

            
Output:

            (7) ["One", "Two", "Three", "Four", "Five", "Six", "Seven"]
JavaScript Array constructor() method

The constructor property returns the function that created the Array prototype. For JavaScript arrays the constructor property returns:
function Array() { [native code] } 

            
Example:

            
            const fruit = ["Apple", "Banana", "Cherry"];
            console.log(fruit.constructor);
